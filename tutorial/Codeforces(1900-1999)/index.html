1901-Educational Codeforces Round 158 (Rated for Div. 2)

B

题意
有一条带子被分成 nnn 个单元格，从左到右编号为 111 到 nnn 。最初，每个单元格中都写有一个整数 000 。

单果与芯片玩游戏。游戏由几个回合组成。在第一轮中，Monocarp 将芯片放入色带的 111 st 单元。除了第一轮之外，在每一轮中，魔卡都会做以下两个动作中的**个：

  • 将芯片移动到下一个单元格（例如，如果芯片在 iii 单元格，则将其移动到 i+1i+1i+1 单元格）。如果芯片在上一格，则无法进行此操作；
  • 选择任意一个 xxx 单元格，将芯片传送到该单元格。可以选择芯片当前所在的单元格。

每回合结束时，写入芯片所在单元格的整数会增加 111 。

Monocarp的目标是在某些回合中使 111 \st单元格中包含整数 c1c_1c1​ ， 222 \nd单元格中包含整数 c2c_2c2​ ，…， nnn \th单元格中包含整数 cnc_ncn​ 。他希望尽可能少地传送芯片。

请帮助 Monocarp 计算他传送芯片的最少次数。

维护一个栈结构，从每一层上去再下来的过程

B(更优版)

对于每一层来讲，肯定有一个上去再下来的过程，
只需要维护每次上去的高度即可，
因为如果中间断了，属于下来了再上去

C

题意
给你一个整数数组 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​ ( 0≤ai≤1090 \le a_i \le 10^90≤ai​≤109 )。在一次操作中，你可以选择一个整数 xxx ( 0≤x≤10180 \le x \le 10^{18}0≤x≤1018 )，并用 ⌊ai+x2⌋\lfloor \frac{a_i + x}{2} \rfloor⌊2ai​+x​⌋ 替换 aia_iai​ （ ⌊y⌋\lfloor y \rfloor⌊y⌋ 表示将 yyy 舍入为最接近的整数）。 ⌊y⌋\lfloor y \rfloor⌊y⌋ 表示将 yyy 舍入为最接近的整数）来替换从 111 到 nnn 的所有 iii 。请注意，每次操作都会影响数组中的所有元素。

打印使数组中所有元素相等所需的最小操作数。

如果操作次数小于或等于 nnn ，则打印每次操作所选择的 xxx 。如果有多个答案，则打印任意一个。

处理最小值和最大值即可
很显然是maxn不断进行(maxn+minn)/2
等价表示：
最大值为偶数，最小值为奇数，(maxn+1)/2,(minn+1)/2
否则为(maxn+0)/2,(minn+0)/2

D

题意
瓦夏是一个与怪物搏斗的巫师。再来有 nnn 只怪物站成一排， iii 只怪物的生命值是 aia_iai​ 。

瓦夏是一个非常强大的巫师，会很多超强的法术。在这场战斗中，他决定使用连锁闪电咒来打败所有怪物。让我们看看这个咒语是如何起作用的。

首先，瓦夏选择某个怪物（ 1≤i≤n1 \le i \le n1≤i≤n ）的指数 iii 和咒语的初始威力 xxx 。然后，咒语会准确命中怪物*** nnn 次，每个怪物命中一次。法术的第一个目标总是怪物 iii 。除了第一个目标之外，连锁闪电会在每个目标中随机选择一个未被法术击中，且与已被击中的怪物相邻的怪物。因此，每个怪物都会被击中一次。第一个被法术击中的怪物会受到 xxx 点伤害，第二个怪物会受到 (x−1)(x-1)(x−1) 点伤害，第三个怪物会受到 (x−2)(x-2)(x−2) 点伤害，以此类推。

瓦夏想展示一下自己有多么强大，所以他想用一个连锁闪电法术杀死所有怪物。如果怪物受到的伤害不低于其健康值，则视为死亡。另一方面，瓦夏想显示出他并不那么在乎，所以他想选择最小的初始法术威力 xxx ，这样就能杀死所有怪物，无论哪只怪物（在能被击中的怪物中）在每一步都被击中。

当然，瓦夏是一名巫师，但确定最佳法术设置所需的计算量远远超出了他的能力范围，所以你必须帮助他找到杀死所有怪物所需的最小法术威力。

请注意，瓦夏选择的是初始目标和法术威力，其他事情应视为随机，瓦夏希望即使在最坏的情况下也能杀死所有怪物。

考虑最坏情况
记录下选择每个点作为起始点，前面和后面的最大值
在这些最大值中取最小值即可
