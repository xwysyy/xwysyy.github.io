类和对象

类和对象的定义

在 C++ 中，类是一种数据类型，它定义了一个对象的属性和行为。而对象是类的一个实例化。一个类可以包含以下内容：

  • 属性：也称为数据成员或字段（field），表示类的状态。
  • 方法：也称为函数成员或操作（operation），表示类的行为。
  • 构造函数和析构函数：用于创建和销毁对象。
  • 访问控制符：用于限制成员的访问权限，包括 public、private 和 protected。

访问控制符

C++ 中的访问控制符包括 public、private 和 protected。它们用于限制成员的访问权限。

  • public：表示公共访问，任何人都可以访问该成员。
  • private：表示私有访问，只有类的成员函数和友元函数可以访问该成员。
  • protected：表示保护访问，派生类可以访问该成员，其他函数无法访问。

重载

函数重载

函数重载指的是可以定义多个函数，它们具有相同的名称但不同的参数列表。根据调用函数时提供的实参类型和数量，编译器可以确定要调用哪个函数。

函数重载的规则

C++ 中对函数重载有以下规则：

  1. 函数名称必须相同。
  2. 参数列表必须不同（包括参数类型、参数数目、参数顺序）。
  3. 函数返回值类型可以相同也可以不同。
  4. 被重载的函数必须在同一个作用域内。

运算符重载

运算符重载指的是可以重新定义或扩展 C++ 中内置运算符的行为。运算符重载以 operator 关键字开头，后面跟着要重载的运算符。

运算符重载的规则

C++ 中对运算符重载有以下规则：

  1. 不能创建新的运算符。
  2. 只能重载已有的运算符。
  3. 不能改变运算符的优先级。
  4. 不能改变运算符的操作数个数。
  5. 不能改变运算符的操作数类型。

STL
